Exercise 6

Write a JavaScript program to check whether a given positive number is a multiple of 3 or 7.
If the number is divisible by 3, then write 'Fizz' in the console,
if the number is divisible by 7, then write 'Buzz' in the console,
and if the number is divisible by 3 and 7, write 'FizzBuzz' in the console.
